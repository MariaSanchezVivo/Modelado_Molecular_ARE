# Modelado_Molecular_ARE
Análisis Dinámico del Péptido ARE (ACE–Ala–Arg–Glu–NME)

Este repositorio contiene el flujo de trabajo completo para el estudio conformacional y termodinámico del tripéptido ARE, desarrollado para la asignatura de Modelado Molecular. 
Incluye datos de simulaciones a 298 K y 400 K, junto con un análisis de alta resolución de 500 ps (298K). 

El proyecto integra el uso de GROMACS para la dinámica molecular, y librerías de Python para el procesamiento estadístico y la generación de figuras del informe final.
